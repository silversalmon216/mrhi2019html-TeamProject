우리가 배운것

c/c++

가장 널리 쓰이는 프로그래밍언어중 하나이다.
다른 언어들보다 빠른 속도를 가지고 있으며 현재도 사용되는 많은 언어들의 기반이 되는 언어이다.

visual studio

MS사에서 개발한 디버깅툴로 C와 C++ 이외에도 여러 언어들의 개발환경을 제공한다.

java

전세계적으로 널리 쓰이고있는 프로그래밍 언어로 객체지향이며 운영체제에 독립적이라는 특징이 존재한다.

eclipse

자바 기반의 통합개발환경으로 어떤 운영체제에서도 동일하게 개발을 진행할수 있도록 지원해준다.

Android

구글에서 리눅스 기반으로 개발한 오픈소스 플랫폼으로 스마트폰이나 태블릿등에서 동작할수 있는 모바일운영체제와 미들웨어 및 중요 어플리케이션이 포함된 소프트웨어 집합이다.

Android Studio

안드로이드개발을 지원하는 통합개발환경으로 자바와 코틀린을 통한 개발을 지원한다.

web

웹페이지의 모습을 정의해주는 마크업언어인 HTML, 디자인적인 표현을 나타내는 CSS, 웹페이지에서의 동작을 구현하는 JS로 구성된어 웹페이지를 제작할 수 있다.

JQuery

웹프로그래밍에서 자바스크립트의 동작을 쉽게 구현해놓은 라이브러리로 적용이 간편하다.

bootstrap

반응형 웹 디자인을 지원하는 JQuery 기반의 오픈소스 프론트엔드 라이브러리로 여러가지 디자인을 CSS와 JavaScript로 만들어놓은 것이다.

Git

분산형 버전관리 시스템(VCS)으로 속도가 빠르고 오프라인환경에서도 작업할수 있다는 장점이 있다.
